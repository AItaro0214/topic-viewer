# アメリカ国民が求める外交とは――安全保障優先と深まる党派間の溝
*2026年3月5日 | Gallup*

## 「テロ防止」と「核拡散阻止」が圧倒的な最優先事項

2026年2月に実施されたギャラップ社の全米調査（成人1,001名、誤差±4ポイント）が、アメリカ国民の外交優先事項を浮き彫りにした。国民は**自国の安全保障と経済的利益**を海外での人道支援や民主化支援よりも明確に優先している。

| 外交目標 | 支持率 |
|---|---|
| 国際テロの防止 | 83% |
| 核兵器拡散の阻止 | 80% |
| エネルギー供給の確保 | 76% |
| NATOとの安全保障協力 | 68% |
| 同盟国の安全保障 | 65% |
| 有利な通商政策の推進 | 64% |
| 世界の貧困・疾病の削減 | 61% |
| 海外での人権促進 | 55% |
| 海外への民主主義構築 | 36% |

- **安全保障と人道のギャップ**: テロ防止（83%）と民主主義構築（36%）の間には47ポイントもの差があり、国民は「世界の警察官」よりも「自国の守護者」としての役割を求めている
- **貿易への楽観**: 82%が貿易を「脅威よりも経済成長の機会」と見ており、この数値は安定的に推移している
- **対外援助への厳しい目**: 42%が対外援助支出を「多すぎる」と回答、「少なすぎる」の31%を上回った

> 国民が望んでいるのは、海外での国家建設ではなく、自国の保護と経済的競争力の確保である。

## 世界における米国のイメージに広がる不安

同じ調査から、アメリカ国民が自国の**国際的地位**に対して矛盾した認識を抱いていることが明らかになった。軍事力には自信を持ちつつも、世界からの評価には悲観的な見方が広がっている。

- **軍事力への自信**: 59%が米国を「世界最強の軍事大国」と認識
- **経済力への懸念**: 米国を「世界一の経済大国」と見る国民はわずか25%
- **国際イメージの悪化**: 59%が米国は世界から「好ましくない目」で見られていると回答（2007年のイラク戦争期と同水準）
- **多国間主義への支持**: 66%が同盟国の支持なしの単独軍事行動に反対

| 指標 | 数値 | 備考 |
|---|---|---|
| 国際的地位への不満 | 61% | 過半数が現状に不満 |
| 軍事的優位の重要性 | 64% | 依然として高い支持 |
| 単独行動への反対 | 66% | 多国間協調を選好 |
| NATO関与の維持・増加 | 77% | 撤退支持はわずか7% |

## 記録的な党派間格差が示すもの

今回の調査で最も注目すべきは、**党派間の認識の溝**が過去最大に達したことだ。米国の国際的地位への満足度において、共和党支持者と民主党支持者の差は実に**73ポイント**に達した。

- **共和党支持者**: 80%が現状に満足、軍事的優位を92%が重視、単独行動を48%が支持
- **民主党支持者**: わずか7%が現状に満足、同盟国との協力を88%が重視、NATO関与の増加を49%が支持（前年比22ポイント増）
- **対外援助**: 共和党の61%が「多すぎる」と回答する一方、民主党では15%にとどまる

> 党派によってまったく異なるアメリカ像が描かれている。共和党は力による秩序を、民主党は協調による安定を志向しており、外交政策の方向性をめぐる国内合意の形成がますます困難になっている。

この調査は米国・イスラエルによるイランへの軍事行動の直前に実施されたものであり、その後の展開が国民の認識をどう変化させるかが注目される。
