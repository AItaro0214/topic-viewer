# 脳科学の最前線：腸内細菌の脳侵入、記憶の捏造、認知症の早期発見
*2026年3月14日 | PsyPost, Neuroscience News*

## 高脂肪食が腸内細菌を脳に送り込む

エモリー大学の研究チームが、高脂肪食がわずか9日間で腸壁を弱め、生きた腸内細菌が迷走神経を通じて直接脳に到達することを発見した。血液を介さない、これまで未知の経路である。

| 条件 | 脳内細菌 |
|------|---------|
| 通常マウス（高脂肪食） | 検出（数百の細胞） |
| 迷走神経切断マウス | 大幅に減少 |
| 血液サンプル | 細菌フリー |
| 通常食に復帰 | 腸壁修復で消失 |

- **経路の特定**: 細菌は脳組織より先に迷走神経で検出——血流経由ではなく神経経路での移動を証明
- **可逆性**: 通常食への復帰で腸壁が修復され、脳内細菌も消失
- **疾患への示唆**: アルツハイマー病や自閉症などの神経疾患が「腸で始まる」可能性を示唆

> 「食事が脳を変える」は比喩ではなく、物理的に細菌が脳に侵入するという文字通りの現象だった。

## 心理学者が「偽の記憶」を意図的に植え付けて記憶の脆弱性を解明

103人の参加者に対し、実際には経験していない出来事の記憶を「ブラインド植え付け法」で生成する実験が行われた。イベントのもっともらしさと提示回数が偽記憶の形成を劇的に左右することが判明した。

| 条件 | 偽信念の形成率 |
|------|-------------|
| もっともらしい事象（1回） | 52% |
| もっともらしくない事象（1回） | 10% |
| もっともらしい事象（反復） | 38.5% |
| もっともらしくない事象（反復） | 22.7% |

- **逆説的発見**: 反復提示は、もっともらしくない事象の信念形成を**増加**させるが、もっともらしい事象では**減少**させる
- **偽記憶vs偽信念**: 鮮明な感覚的記憶の形成率は全条件で9〜16%と低い——信じることと覚えていることは別の現象
- **法的意味**: 誘導的な取調べや証言の信頼性に直接的な影響

> 記憶は録画映像ではなく、社会的文脈によって書き換え可能な物語である。

## テリー・プラチェットの小説が認知症発症10年前に兆候を示していた

ディスクワールドシリーズの全作品を言語分析した研究で、作家テリー・プラチェットの語彙多様性——特に形容詞のバリエーション——が、正式な認知症診断の約10年前から低下し始めていたことが判明した。

- **最初の有意な低下**: *The Last Continent*（診断約10年前に出版）
- **対象**: 後部皮質萎縮症（稀なアルツハイマー型）
- **早期発見の窓**: レカネマブやドナネマブなどの新薬は発症前投与が最も効果的
- **限界**: 教育、職業、執筆習慣も言語に影響するため、単独での診断ツールにはならない

> 文学作品の言語分析が、非侵襲的な認知症早期スクリーニングの新たなツールとなる可能性がある。

## 睡眠薬の大半は睡眠時無呼吸を悪化させない

藤田医科大学が32の臨床試験（1,871人）を統合分析し、不眠症と閉塞性睡眠時無呼吸の合併症（COMISA）患者に対する12種の睡眠薬の安全性を包括的に比較した。初のネットワークメタ分析である。

| 薬剤 | 呼吸安全性 | 睡眠改善効果 |
|------|-----------|------------|
| レンボレキサント | 良好 | 高い |
| ダリドレキサント | 良好 | 高い |
| ゾルピデム | 良好 | 高い |
| テマゼパム | 要注意（酸素飽和度低下） | 中程度 |

- **安全性確認**: 大半の現代的睡眠薬は呼吸パラメータに有意な悪影響なし
- **唯一の例外**: ベンゾジアゼピン系のテマゼパムが睡眠中の動脈酸素飽和度を低下させる可能性
- **個別化治療**: 入眠困難か中途覚醒かに応じた薬剤選択が推奨される

## 脳は目を動かす0.1秒前に映像を「予測」している

脳が眼球運動（サッケード）の直前に視覚像を94%の精度で予測していることが、残像を利用した実験で明らかになった。脳は運動指令の「エフェレンスコピー」を使って視覚シフトを事前に計算している。

- **予測精度**: 約94%（一貫して6%の過小評価＝「ハイポメトリア」）
- **残像の動き**: 網膜に固定された残像が動いて見えるのは、脳の予測計算の結果
- **応用**: VR開発や動揺病の理解に直接的な示唆を提供
