# アルゴリズムと世論：SNSと政治意識の最新動向
*2026年3月3日 | PsyPost, Gallup*

## Xのアルゴリズムが政治的意見を「右」に動かす

*Nature*に掲載された研究が、Xのアルゴリズムフィードがユーザーの政治的意見を体系的に右方向へシフトさせることを実証した。4,965名の米国ユーザーを対象に7週間にわたって実施された実験で、アルゴリズムに触れたグループは共和党寄りの課題を優先し、トランプ関連の調査に対してより懐疑的になった。

- **実験設計**: 参加者をアルゴリズムフィードと時系列フィードにランダムに割り当て
- **効果量**: 0.08〜0.12標準偏差——小さいが統計的に有意
- **不可逆性**: アルゴリズムをオフにしても意見は元に戻らなかった

| 要素 | アルゴリズムの影響 |
|------|------------------|
| 保守的コンテンツ | 積極的に推進 |
| 右派活動家 | 露出を増幅 |
| 従来型ニュースメディア | 表示を抑制 |
| 影響の持続性 | オフ後も残存 |

> 「アルゴリズムはユーザーの情報環境に永続的な足跡を残しうる」— Widmer研究者

- **メカニズム**: ユーザーはアルゴリズム推薦を通じて保守系アカウントをフォローし始め、アルゴリズム停止後もフォローが維持されるため、情報環境が恒久的に変化する
- **限界**: 2023年夏の米国ユーザーのみが対象であり、他のプラットフォームや地域への一般化には注意が必要

## アメリカ人が最も懸念する問題は「政府」

2026年2月のギャラップ世論調査によると、アメリカ人が国の最重要課題として挙げるのは依然として「政府・政治的リーダーシップ」で、回答者の29%がこれを選んだ。移民問題が20%で急上昇し、経済問題が11%で続いた。

- **党派による分断**: 民主党員の44%が政府を最大の問題と回答（歴代最高水準に接近）、一方で共和党員は19%、無党派は24%
- **移民への懸念**: 全政党グループで上昇——共和党32%、民主党18%、無党派15%
- **トランプ政権下の平均**: 政府への懸念は27%で、バイデン政権時の19%を大幅に上回る

| 課題 | 回答率 |
|------|--------|
| 政府・政治リーダーシップ | 29% |
| 移民 | 20% |
| 経済全般 | 11% |
| インフレ | 8% |
| 国の統一 | 7% |
| 司法制度 | 6% |

> 民主党と共和党では「政府が問題」の意味するところが正反対である——一方はトランプ政権を、他方は民主党のリーダーシップを指している。

## アルゴリズムと世論の交差点

これら2つの研究を並べて見ると、現代の政治的意見形成がいかに複雑な力学の中にあるかが浮かび上がる。SNSのアルゴリズムは意図せずして情報環境を歪め、その影響はアルゴリズムを停止した後も持続する。一方で世論調査は、同じ「政府への不満」という言葉の裏に党派ごとにまったく異なる意味が存在することを示している。

- **情報環境の固定化**: アルゴリズムによる情報の偏りは、フォロー行動を通じて自己強化される
- **分断の深化**: 同じ課題について対極の解釈が共存し、対話の基盤が失われつつある
- **民主主義への問い**: 情報のフィルタリングが意見形成に不可逆的な影響を与えうるという知見は、プラットフォーム設計の倫理的責任を問い直す
