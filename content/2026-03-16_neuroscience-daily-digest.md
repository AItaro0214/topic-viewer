# 脳科学の最前線：聴覚の謎、脳インプラント、ADHDと睡眠
*2026年3月16日 | Neuroscience News*

## 騒がしいパーティーで会話を聞き取れる理由が解明された

MITの神経科学チームが、脳が雑踏の中で特定の声を聞き分ける「カクテルパーティー問題」の神経メカニズムを*Nature Human Behavior*で解明した。脳は「乗算的ゲイン」によって標的の声に同調したニューロンの活動を増幅し、周囲のノイズを抑制する。

| 分離の軸 | 脳の性能 |
|---------|---------|
| 水平方向（左右） | 優れている |
| 垂直方向（上下） | 苦手 |
| 声の高さが似た話者 | エラーが増加 |

- **進化的背景**: 水平方向の優位性は、地面レベルの脅威検知に適応した結果
- **計算モデル**: 乗算的ゲインを組み込んだモデルが人間の選択的聴取を再現
- **応用**: 人工内耳の設計改善に直接貢献

> 脳は音声を「聞く」のではなく、数学的に増幅・抑制することで「選んでいる」。

## 恐怖の音への「脳のショートカット」を発見

脳が危険な音に対して意識的な処理より数ミリ秒早く反応する**直通経路**が人間で初めて確認された。下丘から視床の内側膝状体を経由し、基底外側扁桃体に到達する白質結合である。

- **無意識の処理**: 意識が音を認識する前に恐怖反応が始まる
- **逆説的発見**: この経路が強い人ほど恐怖反応が強いが、騒がしい環境での**聴覚弁別能力も高い**
- **臨床的意義**: 不安障害やPTSDでこの回路が過活動になっている可能性

> 恐怖は「考える前に感じる」——この古代の神経経路が私たちを守り続けている。

## 脳インプラントが毎分22語の高速タイピングを実現

マサチューセッツ総合病院とブラウン大学の共同チームが、麻痺患者が指の動きを「試みる」だけで仮想QWERTYキーボードを操作できる脳コンピュータインターフェースを開発した。

| 指標 | 性能 |
|------|------|
| タイピング速度 | 毎分110文字（22語） |
| 単語エラー率 | 1.6% |
| 初期設定 | わずか30文で完了 |
| 対象疾患 | ALS、脊髄損傷 |

- **家庭での実用性**: 2名の臨床試験参加者が自宅で成功裏に使用
- **予測言語モデル**: 精度向上のために統合、既存の視線追跡システムを大幅に上回る速度
- **意義**: 「考えるだけで話す」テクノロジーが現実の生活空間に到達

## ADHDの注意力低下は脳の「居眠り」だった

ADHD成人32人と定型発達者31人を比較した研究で、ADHDの脳では覚醒中に**睡眠様の脳活動**がより頻繁に発生することが判明した。この「局所的睡眠」が注意力の低下、反応時間の遅延、エラー率の増加と直接相関する。

- **現象**: 脳の一部が覚醒中に数秒間「眠る」——全脳的な睡眠ではなく局所的
- **正常でもあるが**: 「睡眠様脳活動は要求の高い課題中に誰でも起こる正常な現象だが、ADHDではその頻度が有意に高い」
- **新治療の可能性**: 睡眠中の特定の聴覚刺激で深い睡眠波を強化し、翌日の局所的睡眠を減少させる非薬物アプローチ

> ADHDの「集中できない」は意志の問題ではなく、脳の一部が文字通り眠っている現象である。

## ペットボトルがパーキンソン病の薬に変わる

エディンバラ大学の研究者が、PETプラスチックをパーキンソン病の主要治療薬**L-DOPA**に変換する遺伝子改変大腸菌を開発した。年間5,000万トン生産されるPETの「アップサイクル」という革新的アプローチである。

- **プロセス**: プラスチック → テレフタル酸 → 大腸菌による生物学的変換 → L-DOPA
- **環境的意義**: 化石燃料ではなく廃棄物を炭素源として使用
- **世界初**: 「プラスチック廃棄物を神経疾患の治療薬に変換した初の自然・生物学的プロセス」

> ゴミから薬へ——環境問題と医療問題を同時に解決する可能性を秘めた画期的研究。
