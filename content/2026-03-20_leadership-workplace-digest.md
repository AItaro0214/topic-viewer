# リーダーシップの盲点：説明責任と「コーポレート・ブルシット」の心理学
*2026年3月20日 | Gallup / PsyPost*

## 説明責任こそリーダーシップ最大の弱点——Gallupが警鐘

7つのコアコンピテンシーの中で、リーダーが最も苦手とするのは何か。ギャラップが2万3,000人以上を対象に実施した調査が、その答えを明確に示している——**説明責任（アカウンタビリティ）の創出**だ。リーダーの自己評価においても、部下のマネージャーによる評価においても、このコンピテンシーが最下位を記録した。

> 「明確な期待値を持つチームは、大幅に高品質な仕事を生み出し、より効率的で収益性も高い」——Gallup研究チーム

| コンピテンシー | マネージャーが「卓越している」と評価した割合 |
|-------------|----------------------------------|
| 関係構築 | 相対的に高い |
| 人材育成 | 中程度 |
| 変革推進 | 中程度 |
| インスピレーション | 中程度 |
| 批判的思考 | 中程度 |
| 明確なコミュニケーション | 中程度 |
| **説明責任の創出** | **30%（最低）** |

- **エンゲージメントへの影響**: 説明責任に卓越したリーダーのもとでは、マネージャーのエンゲージメントが51%に達するのに対し、そうでない場合は17%にとどまる（**3倍の差**）
- **自己認識のギャップ**: リーダーは7つのうち6つのコンピテンシーで自己評価が部下評価より20ポイント以上高い。説明責任だけはそのギャップが最小——**双方が弱さを認識している**
- **調査規模**: 2025年8月、米国成人23,068名を対象（誤差±1.0ポイント、信頼水準95%）

説明責任を機能させるには、年次レビューへの依存から脱却し、目的を具体的な役割期待に落とし込み、**週次コーチング会話**を習慣化することが鍵となる。

## 「コーポレート・ブルシット」を重宝する人ほどリーダーシップが弱い

企業のミッションステートメントや幹部のスピーチに散りばめられた抽象的な「ビジネス用語」——これを印象的だと感じる人ほど、実際の職場判断能力が低いことが明らかになった。米国・カナダの1,018名の就業成人を対象にした4段階の研究で、Corporate Bullshit Receptivity Scale（CBSR）が開発・検証された。

> 「『ブルシット受容性』は文脈に大きく依存する——職業的専門知識や職場環境によって大きく変わる」——Shane Littrell（研究者）

- **相関パターン**: CBSR高得点者は分析的思考・流動的知能のスコアが低く、職場での意思決定テストでも低成績
- **逆説的な傾向**: 同じ高得点者はミッションステートメントを「感動的」と評価し、上司を「変革型リーダー」と見なす傾向が強い
- **最強の予測因子**: CBSRは調査した指標の中で、**職場での意思決定能力の低さを最もよく予測**した

| 尺度 | CBSRとの関係 |
|-----|------------|
| 分析的思考力 | 負の相関 |
| 流動的知能 | 負の相関 |
| 職場意思決定テスト | 負の相関 |
| ミッション文への共感度 | 正の相関 |
| 上司を「変革型」と評価 | 正の相関 |

## 説明責任とブルシットに共通する組織的課題

2つの研究は、現代の職場が抱える同一の病巣を指し示している。**印象的な言葉が具体的な期待値と実行を代替してしまう**という問題だ。

Gallupの研究では、説明責任の欠如がエンゲージメントの低下に直結することが示された。PsyPostの研究では、曖昧な言語そのものが認知能力の低い人々を組織内で優位に立たせる可能性を示唆している。

- **マネジメントへの示唆**: 「コミットメント」「シナジー」「ビジョン」といった言葉を、測定可能な行動指標に変換することが不可欠
- **採用・評価への示唆**: 印象的なプレゼンテーションではなく、具体的な判断・行動・成果を評価基準にする
- **文化醸成への示唆**: 「言葉の巧みさ」よりも「説明責任の文化」が、チームのパフォーマンスと収益性を左右する
