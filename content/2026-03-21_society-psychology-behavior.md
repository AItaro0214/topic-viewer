# 知性・幸福・リーダーシップ――社会科学の最前線

*2026年3月21日 | PsyPost / Gallup / Neuroscience News*

## 知性が「進歩的な価値観」を生むのは大学卒業後だけ

「頭が良い人はリベラルになる」という通説は半分しか正しくない。サウス・アラバマ大学の研究が、より精緻な事実を明らかにした。子供時代の知能スコアと成人後の進歩的な社会態度の相関は、**大学卒業者においてのみ**顕著に現れるという。2つの補完的な研究が実施された。

- **横断研究**: ミネソタ双生児家族研究の中年親世代3,291人で、知能・教育歴・伝統主義（道徳規範・宗教的権威・しつけへの態度）を分析
- **縦断追跡**: 同家族の子世代2,769人を17歳・24歳・29歳の3時点で追跡し、価値観の変容を観察

17歳時点では知能と進歩的な価値観の間に関連は見られなかった。しかし大学教育を受けた高知能者は17〜29歳の間に価値観が大きく変容した。非大学進学者は知能に関わらず、むしろ年齢とともに伝統的な信念が強まる傾向があった。変化は大学院・専門職大学院の参加者でさらに強まった。

> 「高知能者は同調圧力に抵抗するはずだが、それでも大きな価値観の変化を示した。これは教員の授業が直接的な役割を果たしていることを示唆する」――研究チームの分析

## 世界幸福度ランキング：安定と変化が交錯

フィンランドが9年連続で「世界で最も幸せな国」の座を守った。第14回世界幸福度報告書（Gallup世界調査）は136カ国を対象に、生活評価を0〜10点のはしご尺度で3年平均を算出している。

| 順位 | 国名 |
|---|---|
| 1位 | フィンランド |
| 2位 | アイスランド |
| 3位 | デンマーク |
| 4位 | コスタリカ |
| 5位 | スウェーデン |
| 6位 | ノルウェー |
| 7位 | オランダ |
| 8位 | イスラエル |
| 9位 | ルクセンブルク |
| 10位 | スイス |

注目すべきは**コスタリカの4位入り**（中南米として過去最高）と、**カナダ（6位→25位）・米国（17位→23位）** の大幅下落だ。136カ国中79カ国が2006〜2010年比で幸福度を有意に高めた一方、41カ国は有意な低下を示した。最大の上昇国はセルビア・ブルガリア・ラトビア、最大の下落国はアフガニスタン・レバノン・イエメンなどで、紛争・不安定が主因だ。

若者世代の幸福については、世界の約90%をカバーする10地域のうち8地域で若者の生活評価が15年前より向上している。しかし**米国・カナダ・オーストラリア・ニュージーランドの若者は平均0.86ポイント低下**しており、先進経済圏での逆行が課題として浮かび上がった。

## リーダーシップの最大の弱点は「説明責任の創出」

ギャラップが2万3,068人の就業者を対象に実施した調査で、7つのリーダーシップコンピテンシーのうち「説明責任の創出」が最も低い評価を受けた。リーダーが卓越していると評価した管理職はわずか**30%**にとどまる。

| コンピテンシー | 概要 |
|---|---|
| 関係構築 | 信頼ある人間関係の形成 |
| 人材育成 | 部下の成長支援 |
| 変革のリード | 変化への適応と推進 |
| インスパイア | 意欲の喚起 |
| 批判的思考 | 複雑な問題への対処 |
| 明確なコミュニケーション | 期待値の伝達 |
| **説明責任の創出** | **高業績への責任体制の構築（最低評価）** |

説明責任が強いリーダーのチームではエンゲージメント率が**51%**に達するのに対し、弱いチームではわずか**17%**（3倍の差）。またリーダー自身の自己評価と管理職の評価には6コンピテンシーで**20ポイント以上の開き**があり、認知バイアスが自覚的改善を阻害している。

> 「説明責任は年次評価に留まらず、コーチングと認知を統合した週次の対話の中で実践される規律でなければならない」

## デザイナー犬の「気質神話」を科学が否定

コッカプー、カバプー、ラブラドゥードルなどの人気デザイン犬は「親犬のいいとこ取り」と宣伝されることが多い。しかし王立獣医大学（英国）が9,402頭のデータをイヌ行動評価研究アンケート（C-BARQ）で分析した結果、この主張は科学的根拠に乏しいことが判明した。

| ブリード | 問題行動が多かった比較数/全体 | 主な問題行動 |
|---|---|---|
| コッカプー | 16/24 | 飼い主・見知らぬ人への攻撃性、過度の興奮 |
| カバプー | 11/24 | 分離不安、犬への恐怖、興奮性 |
| ラブラドゥードル | 5/24（改善は6） | 比較的良好、攻撃性ではプードルより優秀 |

全体では行動比較の**44.4%**でクロスブリードが親犬種より問題行動が多く、問題が少なかったのはわずか**9.7%**だった。これはデザイン犬が「ハイブリッド強勢」を持つという通説を直接否定するデータだ。

- 研究は *PLOS ONE* に掲載
- サンプルは英国の飼い主9,402人からのデータ
- 行動特性は12カテゴリーにわたって評価

研究チームは、購入前に行動上の課題を理解しておくことが「オーナーと犬双方の福祉」に不可欠だと強調している。デザイン犬の魅力は否定できないが、トレーニングと現実的な期待値の設定が成功する飼育関係の基盤となる。
